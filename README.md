# 🧠 MathGenius AI
### Applied Mathematics Problem & Solution Generator

MathGenius AI is a state-of-the-art web application built with **Next.js 16**, **Google Gemini**, and **OpenAI GPT-4**. It empowers educators, students, and researchers to generate high-quality, academically rigorous applied mathematics problems with detailed, step-by-step solutions.

![Engine](https://img.shields.io/badge/Engine-Gemini_&_GPT-blue)
![LaTeX](https://img.shields.io/badge/Math-LaTeX_Support-green)

## ✨ Features

- **Multi-Provider Support**: Switch between **Google Gemini** and **OpenAI ChatGPT** seamlessly.
- **Expert-Level Generation**: Uses advanced prompting to simulate an expert professor's reasoning.
- **Dynamic Difficulty**: Choose between Easy, Medium, Hard, and Research levels.
- **LaTeX Integration**: Beautifully rendered equations using KaTeX and React-Markdown.
- **Export Capabilities**: Download your problems as high-quality PDF or raw LaTeX files.
- **Smart Workspace**: Toggle solutions, copy to clipboard, and print-ready layouts.
- **API Validation**: Real-time format checking to ensure your API keys are correct.
- **Premium Aesthetics**: Sleek dark mode support with a modern glassmorphism design.

## 🚀 Getting Started

### Prerequisites
- Node.js 18+
- An API Key from [Google AI Studio](https://aistudio.google.com/app/apikey) or [OpenAI](https://platform.openai.com/account/api-keys).


### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/mdsamhussain1996/MathGenius-AI.git
   cd MathGenius-AI
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
4. Open [http://localhost:3000/MathGenius-AI](http://localhost:3000/MathGenius-AI) in your browser.

## 🛠 Tech Stack
- **Framework**: Next.js 16 (App Router)
- **AI SDKs**: @google/generative-ai & openai
- **Styling**: Tailwind CSS 4
- **Icons**: Lucide React
- **Math Rendering**: KaTeX + remark-math + rehype-katex
- **Exports**: html2canvas + jsPDF
